Class: ClientNetworkValue

ClientNetworkValue(initialValue, id, ws, lerpTime, lerpAmount, lerpAccuracy, lerpThreshold)

Client Network Value - A living value that at any point if the server decides was incorrect will re adjust to be correct with the last received correct network value.
